# Team Resume – Team Button Mashers

## Team Overview
Team Button Mashers is a group of web software development students with combined experience in full-stack development, including Java, JavaScript, PHP, HTML, CSS, and SQL databases. The team has worked on multiple database-driven applications and RESTful web services, with a focus on backend development, usability, and scalable design.
